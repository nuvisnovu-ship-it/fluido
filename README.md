# SmoothPlayer 🎬

Reproductor de video Android con suavizado a 60fps por hardware.

## ✅ Funciones
- Reproducción de videos locales (MP4, MKV, AVI...)
- Streams por URL (HTTP, RTMP, M3U8, IPTV)
- Suavizado por decodificador de hardware
- Ajuste automático del refresh rate de pantalla
- Interfaz oscura elegante

## 📲 Cómo obtener el APK

1. Haz fork de este repositorio
2. Ve a la pestaña **Actions**
3. Selecciona **Build APK**
4. Haz clic en **Run workflow**
5. Espera ~5 minutos
6. Descarga el APK desde **Artifacts**
7. Instálalo en tu Android

## ⚙️ Requisitos
- Android 8.0 (API 26) o superior
- Permitir instalación de fuentes desconocidas
